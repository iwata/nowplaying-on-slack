# nowplaying-on-slack

Spotifyで現在聞いている楽曲をSlackのStatusに表示させるアプリ

## Start

```
$ yarn install
$ SLACK_API_TOKEN=xoxp-xxxx-xxxx-xxxx-xxxx npm start
```
